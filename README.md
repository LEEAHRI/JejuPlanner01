#  제주 잇다
제주도 여행 계획을 간편하게 세울 수 있는 플래너 기능을 제공해서 기존 여행 상품 판매 사이트와 차별점을 둔다. 
사용자가 플래너를 작성하며 저장한 장소와 숙소 데이터를 기반으로 사용자별 맞춤 여행 상품을 추천해주며 
상품의 구매율을 높이고 이용자의 만족도를 높인다.
